# 发送给面试官/HR的话术

## 微信/邮件简短版

您好，我是上周一面试视频剪辑岗位的候选人。

面试后我复盘了一下岗位需求，感觉这个岗位不只是剪辑，也会涉及游戏热点发现、活动跟拍、海外资讯整理和内容输出。

所以我基于星游纪/GamePoch 的游戏发行内容场景，做了一个非正式小 Demo，想作为补充材料展示一下我对岗位的理解。

Demo 名称是 **GamePoch Content Copilot**，主要演示如何把热点发现、标题脚本生成、活动内容准备、发布排期和简报整理放到一个工作台里。

这个不是正式产品，只是我的岗位理解和能力补充展示。您有空的话可以简单看一下，谢谢。

## 更克制版本

您好，我是上周一面试视频剪辑岗位的候选人。

面试后我做了一点复盘，并基于对岗位的理解整理了一个非正式 Demo，主要想补充展示我对游戏发行内容工作的理解。

如果方便的话，我想发您看一下。这个 Demo 主要围绕热点发现、内容选题、短视频脚本、活动准备和简报整理这几个方向。

谢谢。

## 发送时建议

- 不要说“我做了一个你们公司可以直接用的系统”，容易显得冒进。
- 可以说“非正式 Demo / 补充作品 / 岗位理解展示”。
- 语气保持克制，不要催结果。
- 如果对方没回复，不要连续追问。
