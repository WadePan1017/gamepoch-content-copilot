# 面试后补充作品说明

## 作品名称

GamePoch Content Copilot

## 一句话说明

这是我基于面试后对星游纪/GamePoch 游戏发行内容岗位的理解，做的一个非正式 Demo，用来展示我对视频剪辑、活动跟拍、内容运营和海外资讯整理工作的理解。

## 为什么做这个

面试后我复盘了一下岗位需求，感觉这个岗位不只是单纯剪视频，还会涉及：

- 发现游戏热点
- 整理海外资讯
- 想平台选题和标题
- 写短视频脚本
- 准备活动跟拍素材
- 做日报或内部汇报

所以我做了这个小 Demo，想补充展示我对岗位的理解，以及我希望能为团队带来的内容效率提升思路。

## Demo 主要功能

- 今日内容工作台：查看当天内容机会、短视频选题、海外资讯和风险提醒
- 热点发现：从精选内容、Steam 和海外资讯中整理值得跟进的选题
- 内容生成：输入游戏名，生成 B站/抖音/小红书/公众号标题和短视频脚本
- 活动规划：为 Steam Next Fest、Gamescom 等活动生成拍摄清单和采访问题
- 7日发布排期：把热点和活动准备转成平台任务、负责人和素材需求
- 作战简报：导出 Markdown / Word 简报，用于日报、周报或内部汇报
- 演示话术：整理这个 Demo 的讲解流程和岗位价值

## 非正式说明

这个 Demo 不是星游纪/GamePoch 官方系统，也不是正式商业产品，只是我作为候选人基于面试理解做的补充作品。

我希望通过它表达：

- 我理解游戏发行内容团队的真实工作流
- 我不只是做剪辑执行，也愿意主动思考内容效率
- 我能把 AI 工具、信息整理和内容生产结合起来
- 如果有机会加入团队，我可以继续把这类工具和流程完善成真正可用的工作方法
