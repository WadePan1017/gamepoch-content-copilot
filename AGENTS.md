# 项目背景

这是一个为游戏发行公司（星游纪 Gamepoch）开发的内部工具，用于自动化游戏行业数据采集和报表生成。

## 核心需求

公司需要：
1. 每日监控Steam/IGN/GameSpot等行业数据
2. 跟踪游戏发行项目进度
3. 分析竞品游戏的市场表现
4. 生成舆情报告给索尼等合作伙伴

## 技术决策

- 使用FastAPI是因为需要异步调用多个外部API
- 使用Tailwind CSS是为了快速开发UI
- 使用python-docx生成Word报告是因为需要给合作伙伴看正式文档

## 当前状态

工具已基本可用，但需要优化：
- UI需要美化
- 搜索功能需要调试（用户反馈有时不响应）
- 新闻抓取源需要增加稳定性
- 报告内容需要更丰富
